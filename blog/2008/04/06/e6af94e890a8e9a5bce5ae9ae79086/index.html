比萨饼定理

Apr 6th, 2008

    比萨饼（PIZZA）是一种由特殊的饼底、乳酪、酱汁和馅料做成的具有意大利风味的食品，在本文中你只要认识到它是圆的，薄的，均匀的就可以了，而不必关心它的味道如何。有些爱吃比萨饼的人发现并证明了一些定理，并把它们命名为比萨饼定理（PIZZA Theorem）。现介绍如下：

    先说可爱的比萨饼第二定理：The volume of a circular pizza of thickness a and radius z is, of course, pizza. 即：一个厚度为a，半径为z的圆的比萨饼的体积是Pizz*a,也就是Pizza。

    比萨饼第一定理，也叫比萨饼定理，它提供了一个为n个人平分一块比萨饼的方法：在比萨饼上找一随机点P，用通过这个随机点的2n刀把饼分成角度相同的4n份，按顺序编号。第k (k = 1..n) 个人取编号为k，n+k,2n+k,3n+k的分块。那么每个人得到同样多的比萨饼。

简单证明如下：令alpha = Pi/(2*n),r(theta) 表示极坐标中随机点P到边缘的长度。那么第k个人得到的面积有：

    int( (1/2)(r(theta)2 + r(theta + Pi/2)2 + r(theta + Pi)2 + r(theta + 3Pi/2)2 ), theta = (k -1)alpha..kalpha );

下面将证明积分的部分(1/2)(r(theta)2 + r(theta + Pi/2)2 + r(theta + Pi)2 + r(theta + 3Pi/2)2 )等于 2*R2;

    令 r(theta) = a, r(theta + Pi/2) = b, r(theta + Pi) = c, r(theta + 3*Pi/2) = d, 它们边缘上的点分别为A，B，C，D。PA = a, PB = b, PC = c, PD = d.作OM垂直BD于M，ON垂直AC于N，则

    R2 = OD2 = OM2 + MD2 = NP2 + MD2 = ((a - c)/2)2 + ((b + d)/2)2;

    R2 = OC2 = ON2 + NC2 = MP2 + NC2 = ((b - d)/2)2 + ((a + c)/2)2;

两式相加得

    2R2 = (1/2)(a2 + b2 + c2 + d2);

于是第k个人得到的面积为

    int(2R2, theta = (k -1)alpha..kalpha ) = PiR2/n;

证毕。

所以每个人分的的面积相同。
